# Web (Next.js)

Mobile-first civic petition UX optimized for conversion.

## Pages
- `/` landing with hero + trending petitions
- `/petitions/[id]` conversion-first petition page with sticky signature box
- `/create` start petition page
- `/auth/login`, `/auth/signup`
- `/dashboard`
- `/admin`

## UX Rules Implemented
- Single dominant CTA per page
- Sticky signature box with count, progress, and sign form
- Social proof: signed today + progress
- Post-sign share modal: WhatsApp, Facebook, copy link
- Sticky mobile bottom sign CTA
