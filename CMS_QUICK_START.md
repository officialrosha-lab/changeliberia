# CMS Block Editor - Quick Start (5 Minutes)

## Login & Navigate

1. **Log in** to the admin dashboard
2. Click **Admin** in the top navigation
3. Click the **CMS** tab
4. You'll see the **Block-Based Page Editor** section

## Your First Edit (Try This Now!)

### Step 1: Select a Page (30 seconds)
- Find the "Page:" dropdown at the top
- Click it and select **"About"**
- Wait for the page to load (you'll see existing blocks below)

### Step 2: Preview Current Content (30 seconds)
- Look at the right side of the screen - you'll see a preview
- Scroll down in the preview to see all the blocks
- These are your current 5 blocks on the About page

### Step 3: Edit Existing Block (1 minute)
- Find the first block (Hero - "Welcome to Change Liberia")
- Look at the "Edit Block" form below it
- Change the **title** from "Welcome to Change Liberia" to something else (e.g., "Empowering Liberia")
- Scroll down and click the **"Update Block"** button
- ✨ Watch the preview on the right update instantly!

### Step 4: Add New Block (2 minutes)
1. Scroll to the bottom where it says **"Add Block"**
2. Click the dropdown and select **"Text"** (if not already selected)
3. Fill in:
   - **Body:** "This is my new content block. It can contain any text I want."
   - Leave **Title** empty if you like
   - Keep **Alignment** as "left"
4. Click **"Add Block"**
5. ✨ Your new block appears in the preview!

### Step 5: Delete the Block (30 seconds)
- Find your new block in the list
- Click the **Delete** button (red X icon)
- The block disappears from preview instantly

## That's It! You Know the Basics

You just:
- ✅ Selected a page
- ✅ Previewed content
- ✅ Edited a block
- ✅ Added a new block
- ✅ Deleted a block

## Block Types Cheat Sheet

When adding blocks, here are your 9 options:

| Type | Use For | Example |
|------|---------|---------|
| **Hero** | Page headers/big banners | Title + subtitle + button |
| **Text** | Paragraphs and descriptions | Body text, explanations |
| **Image** | Pictures | Photos, screenshots |
| **Grid** | Organized columns | Features, values, steps (1-4 columns) |
| **CTA** | "Sign up now" sections | Calls to action with buttons |
| **Testimonial** | Quotes | "This platform is awesome!" - John |
| **Divider** | Spacing/separators | Just a visual break |
| **FAQ** | Questions & answers | Q: "How do I...?" A: "Click..." |
| **Features** | Feature lists | With icons and descriptions |

**Most Used:** Text, Hero, Grid, FAQ  
**Try First:** Text (easiest)

## Quick Tips

### ✨ Things That Happen Automatically
- ✅ Preview updates instantly as you edit
- ✅ Changes save to the database when you click Update/Add
- ✅ Public pages update instantly (no "publish" button needed)
- ✅ All changes are timestamped in the database

### 💡 Pro Tips
- **Edit in one tab, preview in another:** Open `/about` in another browser tab to see real-time updates
- **Mobile preview:** Use your phone to visit the site and see mobile rendering
- **Copy-paste:** If you want similar content, edit an existing block and reuse the text

### ⚠️ Important
- **No undo:** Changes are saved immediately, so be careful with deletes
- **Admin-only:** Only logged-in admins can access this editor
- **Public pages:** Your edits show up immediately on public pages

## Common Tasks

### Add a New Section to About Page
1. Go to **Admin → CMS**
2. Select **About**
3. Scroll to bottom
4. Add Block → Select **"Text"**
5. Type your section heading in **Title**
6. Type your content in **Body**
7. Click **Add Block**

### Change a Button Text
1. Find the **Hero** or **CTA** block with the button
2. Edit the form (you'll see button fields)
3. Change **Button Text** from "Get Started" to "Learn More" (or whatever)
4. Change **Button Link** if needed (e.g., `/apply` or `/help-center`)
5. Click **Update Block**

### Reorganize Blocks (Current Limitation)
Blocks appear in order 1, 2, 3, etc. Currently you need to:
- Delete blocks in the order you want changed
- Re-add them in the new order

*Future enhancement: Drag-and-drop reordering*

### Temporarily Hide Content
- Delete the block (you can always add it back)
- Or edit it and leave all fields empty (saves as "hidden" content)

## Troubleshooting

### "I don't see my changes on the website"
- **Wait 5 seconds** - sometimes there's a slight delay
- **Refresh the page** - try F5 or Cmd+R
- **Check another browser** - make sure it's not a browser cache issue

### "I accidentally deleted something"
- Check the database backups (contact your developer)
- You can recreate blocks with the same text

### "The page looks weird on mobile"
- This is usually CSS (styling), not content
- Contact your developer
- In the meantime, edit the text to be shorter

### "I can't find the CMS tab"
- Make sure you're logged in as admin
- Make sure you clicked **Admin** first
- Then click the **CMS** tab
